mapController
controller in module map

Description

Controller takes care of the navigation map. Note: The controller gets called before map.main.js

Methods

  • resetColor(index)

    Used for resetting color flag. The color flag determines whether the previous attempted lesson tag gets highlighted in the HUD.

    Parameters
    ParamTypeDetails
    indexint

    The index of the current element in the lessons array

    Returns
    boolean

    True if the flag is successfully reset else false

  • closeDemo()

    ends the demo on the navigation map by closing the demo modal.

    Returns
    boolean

    True every time

  • getNodeProperty(propertyString)

    Finds node property values in localstorage and then returns them

    Parameters
    ParamTypeDetails
    propertyStringstring

    A string to define the property of the node. It can take the values 'x', 'y', 'width', 'height', 'node', 'type'

    Returns
    string

    Value of the property stored in localStorage. If nothing is found value 0 is returned

  • openDemo()

    starts the demo on the navigation map by triggering the demo modal.

    Returns
    boolean

    True every time

Properties

  • setAnimateStarFlag

    Initializes animateStar flag in localStorage
